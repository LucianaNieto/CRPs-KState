CRP County-Level Analysis – Kansas and the Great Plains

This project explores and visualizes data from the USDA’s Conservation Reserve Program (CRP) at the county level, with a focus on Kansas and the broader Great Plains region.

Objectives

Extract and clean publicly available CRP data
Perform exploratory analysis and generate spatial visualizations
Build an interactive app to explore CRP enrollment by year and location
Structure

data/: raw, cleaned, and spatial datasets
scripts/: data processing and analysis scripts
app/: code for the interactive visualization app
docs/: supporting documentation and metadata
reports/: final technical documents and reports
resources/: supporting documents about CRP program from FSA website
